{ images }

canDoConfiguration.images = { url: string }

Description: By defining images here you can tell canDo to load those images before starting the animation.

{
	url: string
}

Example

var translate = CanDo(document.getElementById('translate'), {
    images: {
    	{url: '/images/myImage.png'}
    }
});

Details

url: string

The path to the image.
