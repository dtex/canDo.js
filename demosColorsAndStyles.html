Colors and Styles Demos

Coontrols patterns and gradients for strokes and fills.

context.canDo('createLinearGradient', { args })

Wraps context.createLinearGradient ( x0, y0, x1, y1 )

Creates a linear gradient from point (x0,y0) to (x1, y1).

Returns a gradient object.

This text is displayed if your browser does not support HTML5 Canvas.

context.canDo('createRadialGradient', { args })

Wraps context.createRadialGradient ( x0, y0, r0, x1, y1 r1 )

Creates a radial gradient starting at point (x0,y0) with radius (r0) to end point (x1, y1) with radius r1.

Returns a gradient object.

This text is displayed if your browser does not support HTML5 Canvas.
