{ cuePoints }

canDoConfiguration.cuepoints = { cuePointName: cuePointTime }

Description: A cuePoint is a handle to a point in time on your animation.

{
	cuePointName: cuePointTime
}

Example

var translate = CanDo(document.getElementById('translate'), {
    cuePoints: {
    	'slower': 0.5
    }
});

// Sets the cuepoints on the translate object. Yes, you can change 
the cuepoints at any time.

Details

cuePointName: string

The name of the cuepoint. Any valid JS variable name will do.


cuePointTime: float

Where on our timeline the cuePoint is referring to. This should be a value 
between 0.0 and 1.0
