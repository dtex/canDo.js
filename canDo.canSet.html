CanDo.canSet()

CanDo.canSet( property, [ { propertyKeyframes } ] )

Description: Wraps setters from the HTML Canvas 2D Context.

CanDo.canSet( property, [ { propertyKeyframes } ] )


property  A string containing the name of the built in Canvas property 
you wish to set.

propertyKeyframes A list of keyframes with values to pass to the selected setter. 
These values will be tweened by canDo.

Example

this.canSet('shadowBlur', [
	{ params: [3], cuePoint: 0},
	{ params: [20], cuePoint: 0.5},
	{ params: [3], cuePoint: 1.0}
]);
