canDo.getCurrentKeyframe()

CanDo.getCurrentKeyframe( [ { keyframes } ] )

Description: Grabs the current from and to keyframes based on the current time in canDo.s.time

CanDo.getCurrentKeyframe( [ { keyframes } ] )


{ Keyframes }  A keyframe object that consists of parameters, a cue point/time 
value and optionally an easing function.

Example

this.getCurrentKeyframe( [ 
	{ params: [3], cuePoint: 0},
	{ params: [20], cuePoint: 0.5},
	{ params: [3], cuePoint: 1.0}
] );
