CanDo()

CanDo( canvasElement, canDoConfiguration )


Returns: A canDo wrapped 2d rendering context.

Description: Wraps a canvas element and adds methods and properties for building simple animations.

CanDo( canvasElement, canDoConfiguration )


canvasElement  An HTMLCanvasElement.

canDoConfiguration A set of key/value pairs that configure the canDo animation.

When you create the elements you pass through all the properties you need to configure you animations timeline and paint routines. Take a good look at the canDoConfiguration object as it's an important part of any canDo animation. Also note that a canDo wrapped canvas element still has all of the native methods and properties of the 2dRenderingContext.

Example

var translate = CanDo(document.getElementById('translate'), {
    frameRate: 60,
    duration: 3000,
    easing: 'easeInOutQuad',
    paint: function() {
        this.identity();
        this.clearRect(0, 0, 320, 320);
        this.canDo('translate', [ 
        	{params: [0, 0],cuePoint: 0},
        	{params: [110, 110],cuePoint: 1.0}
        ]);
        drawJSLogo(this);
    }
});
