CanDo.play( {canDoConfiguration} )

CanDo.play( {canDoConfiguration} )

Description: Start the animation if it is not already running with the option to pass in a
{ canDoConfiguration } object.

CanDo.play( {canDoConfiguration} )


canDoConfiguration  A set of key/value pairs that configure the canDo animation.

The most common use case for this function is to trigger a play event while passing in a playback speed.

Example

demo.play( {speed: 1.0} ); // Play forward

- or -

demo.play( {speed: -1.0} ); // Play backwards
