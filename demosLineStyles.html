Line Style Demos

Line style properties give you control over thickness, miter properties and end caps.

context.canSet('lineWidth', { args })

Wraps context.lineWidth [ = value ]

Sets the line width value that is applied to all lines and paths.

This text is displayed if your browser does not support HTML5 Canvas.
