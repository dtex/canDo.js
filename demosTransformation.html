Transformation Demos

Canvas has a series a matrix transformations that can be applied to your rendering context. These transformations are additive so you will usually want to reset the canvas to its default state (the identity matrix) before each call to your paint function.

context.canDo('scale', { args })

Wraps context.scale(x, y)

Applies a scaling transformation to the canvas coordinate grid.

This text is displayed if your browser does not support HTML5 Canvas.

context.canDo('rotate', { args })

Wraps context.rotate( radians )

Applies a rotation to the canvas coordinate grid.

This text is displayed if your browser does not support HTML5 Canvas.

context.canDo('translate', { args })

Wraps context.translate( x, y )

Applies a translation to the canvas coordinate grid.

This text is displayed if your browser does not support HTML5 Canvas.

context.canDo('setTransform', { args })

Wraps context.setTransform( a, b, c, d, e, f )

Sets the current transormation matrix.

This text is displayed if your browser does not support HTML5 Canvas.
