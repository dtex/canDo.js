CanDo.update( {canDoConfiguration} )

Description: Update the animation status by passing in a
{ canDoConfiguration } object.

CanDo.update( {canDoConfiguration} )


canDoConfiguration  A set of key/value pairs that configure the canDo animation.

This function updates our playback head on the timeline, calculates eased time, etc. Generally this method is only called from within canDo, but it's a good way to pass in updates to the playback position.

Example

demo.update( {time: 1.0} ); // Jump to the end of the animation
