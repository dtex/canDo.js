Cando.js

A canvas wrapper for lightweight animations. Ideal for simple animated UI elements in the 2d rendering context.

Download this project as a .zip file Download this project as a tar.gz file

More info: http://dtex.github.com/canDo.js/.

canDo.js is a canvas wrapper for simple animations. It is ideal for animating UI elements (i.e. buttons with fancy hover states). In a nutshell, canDo.js proxies the methods of the HTML Canvas 2D Context. It allows you to define a timeline, keyframes and easing functions.
