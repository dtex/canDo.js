canDo.js Introduction

canDo.js is a canvas wrapper for lightweight animations. It gives you a timeline, cupoints, tweening, basic events and easing functions. It is ideal for animating simple UI elements (i.e. buttons). In a nutshell, canDo.js proxies the methods of the HTML Canvas 2D Context and allows you to add keyframes to the properties you pass in. canDo.JS handles the rest.

You are still free to use the non-proxied built-in functions when appropriate.
