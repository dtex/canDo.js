canDo.js

A canvas wrapper for lightweight animations. It is ideal for animating simple UI elements (i.e. buttons, and little graphical widgets). In a nutshell, canDo.js proxies the methods of the HTML Canvas 2D Context, and allows you to add keyframes and easing functions to the properties you pass in. canDo.JS handles the rest of the animation for you.

canDo.js is:

  • Framework agnostic – It works with jQuery, Prototype.js, etc.
  • Standalone – There are no additional requirements (just 1 file)
  • Tiny – Just x.xKb minified and x.xKb gzipped

canDo.js is not:

  • A scene graph manager
  • A game engine
  • A drawing tool
