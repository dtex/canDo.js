<canvasElement>

canDo( canvasElement, canDoConfiguration )

Description: An HTML Canvas element as found in the HTML 5 Specification.

You can learn more about Canvas elements at these web sites:

  • Mozilla Developer Network
  • HTML5 Canvas Tutorials

interface HTMLCanvasElement : HTMLElement {
	attribute unsigned long width;
	attribute unsigned long height;

	DOMString toDataURL(in optional DOMString type, in any... args);
	void toBlob(in FileCallback, in optional DOMString type, in any... args);

	object getContext(in DOMString contextId, in any... args);
};

Example

<canvas id="demo" width="400" height="360">
	This text is displayed if your browser does not support HTML5 Canvas.
</canvas>
