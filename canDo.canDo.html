CanDo()

CanDo.canDo( contextMethod, [ { methodKeyframes } ] )

Description: Wraps methods from the HTML Canvas 2D Context.

CanDo.canDo( contextMethod, [ { methodKeyframes } ] )


method  A string containing the name of the built in canvas method you wish to call.

methodKeyframes A list of keyframes with values to pass to the selected method. 
These values will be tweened by canDo.

Example

this.canDo('translate', [ 
	{params: [0, 0],cuePoint: 0},
	{params: [110, 110],cuePoint: 1.0}
]);
