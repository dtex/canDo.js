Notes

These demos only show the minimal implementation required to show a particular wrapped function in use. You'll want to look at the full documentation. It's short and sweet, but worth a read.

In the demos I make repeated use of four functions which are called from but not shown in the demo code:

  • void drawJSBox ( canvas2dContext ) - Draws the yellow box in the JS logo
  • void drawJ ( canvas2dContext ) - Draws the "J" in the JS logo
  • void drawS ( canvas2dContext ) - Draws the "S" in the JS logo
  • void drawJSLogo ( canvas2dContext ) - Draws the JS logo by calling the three previous functions
