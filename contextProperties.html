Notes

You can set nearly every 2d context property through canDo. Below is a table listing all the properties, their signatures and wether any value is actually added by canDo. I've also provided links to W3C documentation and examples on jsFiddle where I can.

This list of proeprties is based on W3C Working Draft 25 from May 2011

Properties from HTML Canvas 2D Context

Method Signature Notes Example
Compositing
context.globalAlpha [ = value ]
context.globalCompositeOperation [ = value ]
Colors and Styles
context.strokeStyle [ = value ]
context.fillStyle [ = value ]
Line Styles
context.lineWidth [ = value ]
context.lineCap [ = value ]
context.lineJoin [ = value ]
context.miterLimit [ = value ]
Shadows
context.shadowColor [ = value ]
context.shadowOffsetX [ = value ]
context.shadowOffsetY [ = value ]
context.shadowBlur [ = value ]
Text
context.font [ = value ]
context.textAlign [ = value ]
context.textBaseline [ = value ]
Pixel Manipulation
imagedata = context.createImageData(sw, sh)
imagedata = context.createImageData(imagedata)
imagedata = context.getImageData(sx, sy, sw, sh)
